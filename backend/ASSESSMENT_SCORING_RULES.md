# 呼吸道感染评估系统 - 评分规则汇总

本文档整理了系统中所有8个评估表的详细评分规则。

## 1. CURB-65评分（社区获得性肺炎严重程度评估）

### 评分项目（每项1分，总分0-5分）
| 评分项 | 标准 | 分值 | 数据来源 |
|--------|------|------|----------|
| **C**onfusion（意识障碍） | 存在意识障碍、意识不清、昏迷、嗜睡、意识模糊、神志不清 | 1分 | physical_examination_detail.general_condition |
| **U**rea（尿素氮） | BUN > 7 mmol/L | 1分 | patient.blood_urea_nitrogen |
| **R**espiratory rate（呼吸频率） | ≥ 30次/分 | 1分 | physical_examination_detail.respiration |
| **B**lood pressure（血压） | 收缩压 < 90 mmHg 或 舒张压 ≤ 60 mmHg | 1分 | physical_examination_detail.systolic_bp/diastolic_bp |
| Age（**65**岁） | 年龄 ≥ 65岁 | 1分 | patient.age |

### 风险分级
- **0-1分**：低风险（门诊治疗）
- **2分**：中风险（考虑住院）
- **3-5分**：高风险（住院治疗，考虑ICU）

## 2. PSI评分（肺炎严重程度指数）

### 评分项目（总分可达170+分）

#### 人口统计学因素
| 项目 | 评分规则 | 分值 |
|------|----------|------|
| 年龄 | 男性=年龄值；女性=年龄值-10 | 变量 |
| 性别 | 女性 | -10分 |

#### 基础疾病（每项+分）
| 疾病 | 分值 | 备注 |
|------|------|------|
| 肿瘤性疾病 | +30分 | 当前未实现 |
| 肝脏疾病 | +20分 | 当前未实现 |
| 充血性心力衰竭 | +10分 | 当前未实现 |
| 脑血管疾病 | +10分 | 当前未实现 |
| 肾脏疾病 | +10分 | 当前未实现 |

#### 体格检查
| 项目 | 标准 | 分值 | 数据来源 |
|------|------|------|----------|
| 精神状态改变 | 现病史包含"精神差" | +20分 | patient.present_illness |
| 心率 | > 125次/分 | +20分 | physical_examination_detail.pulse |
| 呼吸频率 | > 30次/分 | +20分 | physical_examination_detail.respiration |
| 收缩压 | < 90 mmHg | +20分 | physical_examination_detail.systolic_bp |
| 体温 | < 35℃ 或 > 40℃ | +15分 | physical_examination_detail.temperature |

#### 实验室检查
| 项目 | 标准 | 分值 | 数据来源 |
|------|------|------|----------|
| 动脉血pH | < 7.35 | +30分 | patient.arterial_ph |
| 血尿素氮 | > 30 mg/dl | +20分 | patient.blood_urea_nitrogen |
| 血钠 | < 130 mmol/L | +20分 | 当前未实现 |
| 血糖 | > 14 mmol/L | +10分 | 当前未实现 |
| 红细胞压积 | < 30% | +10分 | 当前未实现 |
| PaO2 | < 60 mmHg | +10分 | patient.arterial_po2 |

#### 影像学检查
| 项目 | 标准 | 分值 | 数据来源 |
|------|------|------|----------|
| 胸腔积液 | CT显示胸腔积液 | +10分 | physical_examination_detail.ct_imaging_findings |

### 风险分级
- **Ⅰ级**（≤50分）：极低风险，门诊治疗
- **Ⅱ级**（51-70分）：低风险，门诊治疗
- **Ⅲ级**（71-90分）：低风险，门诊或短期住院
- **Ⅳ级**（91-130分）：中风险，住院治疗
- **Ⅴ级**（>130分）：高风险，ICU治疗

## 3. CPIS评分（临床肺部感染评分）

### 评分项目（简化版，总分0-5分）
| 项目 | 评分标准 | 分值 | 数据来源 |
|------|----------|------|----------|
| 体温 | 36.5-38.4℃：0分<br>38.5-38.9℃：1分<br>≥39℃或≤36℃：2分 | 0-2分 | physical_examination_detail.temperature |
| 白细胞计数 | 数据不可用 | 固定0分 | - |
| 分泌物 | 主诉包含"咳痰"：1分<br>否则：0分 | 0-1分 | patient.chief_complaint |
| 氧合指数 | ≥240：0分<br>＜240且无ARDS：2分 | 0-2分 | patient.arterial_oxygenation_index |
| 胸片浸润影 | 数据不可用 | 固定0分 | - |
| 痰培养 | 数据不可用 | 固定0分 | - |

### 风险分级
- **≤6分**：低风险（VAP可能性小）
- **>6分**：高风险（VAP可能性大）

**注意**：当前系统最高只能得5分，所有患者均为低风险

## 4. qSOFA评分（快速序贯器官功能衰竭评估）

### 评分项目（每项1分，总分0-3分）
| 项目 | 标准 | 分值 | 数据来源 |
|------|------|------|----------|
| 意识障碍 | 主诉或现病史包含：意识障碍、嗜睡、昏迷、神志不清等 | 1分 | patient.chief_complaint/present_illness |
| 呼吸频率 | ≥ 22次/分 | 1分 | physical_examination_detail.respiration |
| 收缩压 | ≤ 90 mmHg | 1分 | physical_examination_detail.systolic_bp |

### 风险分级
- **<2分**：低风险
- **≥2分**：高风险（脓毒症高风险，需进一步评估）

### 严重程度分级
- **0-3分**：正常或轻微
- **4-7分**：轻度器官功能障碍
- **8-11分**：中度器官功能障碍
- **12-14分**：重度器官功能障碍
- **≥15分**：极重度器官功能障碍

## 6. 重症肺炎诊断

### 主要标准（满足1项即可诊断）
| 标准 | 数据来源 |
|------|----------|
| 需要机械通气 | patient.ventilator_used = 1 |
| 感染性休克需要血管活性药物 | physical_examination_detail中MAP<65且使用血管活性药物 |

### 次要标准（满足3项即可诊断）
| 标准 | 数据来源 |
|------|----------|
| 呼吸频率 ≥30次/分 | physical_examination_detail.respiration |
| 氧合指数 ≤250 | patient.arterial_oxygenation_index |
| 多肺叶浸润 | CT影像所见包含"多发"、"双肺"、"两肺"等 |
| 意识障碍/定向障碍 | 主诉或现病史包含相关关键词 |
| 血尿素氮 ≥7.14 mmol/L | patient.blood_urea_nitrogen |
| 收缩压 <90 mmHg（需要积极液体复苏） | physical_examination_detail.systolic_bp |
| 白细胞 <4×10^9/L | patient.white_blood_cell_count |
| 血小板 <100×10^9/L | patient.platelet_count |
| 体温 <36℃ | physical_examination_detail.temperature |

### 诊断结果
- **重症肺炎**：满足1个主要标准或3个次要标准
- **非重症肺炎**：不满足上述标准

## 7. COVID-19重型诊断

### 成人重型标准（满足任一项）
| 标准 | 数据来源 |
|------|----------|
| 呼吸窘迫，呼吸频率 ≥30次/分 | physical_examination_detail.respiration |
| 静息状态下，指氧饱和度 ≤93% | patient.blood_oxygen_saturation |
| 氧合指数 ≤300 mmHg | patient.arterial_oxygenation_index |
| 肺部影像学显示24-48小时内病灶明显进展 >50% | 需人工判断，系统暂不支持 |

### 儿童重型标准（需同时满足）
1. 持续高热超过3天
2. 满足以下任一项：
   - 呼吸窘迫（呼吸频率增快）
   - 静息状态下指氧饱和度 ≤93%
   - 氧合指数 ≤300 mmHg
   - 辅助呼吸（鼻翼扇动、三凹征）
   - 嗜睡、惊厥
   - 拒食或喂养困难，有脱水征

### 诊断结果
- **重型**：满足成人或儿童标准
- **非重型**：不满足上述标准

## 8. COVID-19危重型诊断

### 危重型标准（满足任一项）
| 标准 | 数据来源 |
|------|----------|
| 呼吸衰竭，需要机械通气 | patient.ventilator_used = 1 |
| 出现休克 | MAP<65 或使用血管活性药物 |
| 合并其他器官功能衰竭需ICU监护治疗 | patient.intensive_care_unit = 1 |

### 严重程度分级
- **非危重型**（0项）：不满足任何标准
- **危重型**（1项）：满足1个标准
- **危重型（高危）**（2项）：满足2个标准
- **危重型（极危重）**（≥3项）：满足所有标准

## 数据来源说明

### 主要数据表
1. **patient_info**：患者基本信息（年龄、性别、生化指标等）
2. **physical_examination_detail**：体检详细信息（生命体征、CT影像等）
3. **patient_examination_relation**：患者与体检信息关联表

### 关键字段映射
- 体温：physical_examination_detail.temperature
- 血压：physical_examination_detail.systolic_bp/diastolic_bp
- 心率：physical_examination_detail.pulse
- 呼吸频率：physical_examination_detail.respiration
- 氧合指数：patient.arterial_oxygenation_index
- 血氧饱和度：patient.blood_oxygen_saturation
- 尿素氮：patient.blood_urea_nitrogen
- 血小板：patient.platelet_count
- 白细胞：patient.white_blood_cell_count
- CT影像：physical_examination_detail.ct_imaging_findings
- 机械通气：patient.ventilator_used
- ICU监护：patient.intensive_care_unit
- 血管活性药物：dopamine_used, epinephrine_used等

## 使用说明

1. **批量评估**：系统支持对所有患者进行批量自动评估
2. **单个评估**：支持对指定患者进行单独评估
3. **重复评估**：系统会自动跳过已评估的患者
4. **数据完整性**：部分字段缺失时，相关项目记为0分
5. **评估顺序**：建议先进行数据迁移，再执行评估，确保使用最新的结构化数据

## 注意事项

1. **CPIS评分**：由于数据限制，当前系统简化版最高5分，建议调整风险分级阈值
2. **基础疾病**：PSI评分中的基础疾病项目当前未实现，需要额外的病史数据支持
3. **动态评估**：部分标准（如影像学进展）需要时间序列数据，系统暂不支持
4. **人工复核**：建议对高风险患者进行人工复核，特别是涉及意识状态的评估项目
