# src/views 目录说明

本目录包含项目的主要业务页面，按功能模块划分如下：

- **dashboard/**
  - 数据仪表盘页面，包括监控大屏、工作台等。
- **exception/**
  - 异常页面，如 403、404、500 等错误提示。
- **form/**
  - 各类表单页面，支持分组表单、分步表单等复杂场景。
- **list/**
  - 列表页面，支持卡片式和表格式两种展示方式。
- **login/**
  - 登录页面及相关组件。
- **not-found/**
  - 未找到页面，通常用于路由兜底。
- **profile/**
  - 个人信息页，展示用户资料、操作日志等。
- **redirect/**
  - 路由重定向页面。
- **result/**
  - 结果页，展示操作成功或失败的提示。
- **user/**
  - 用户中心相关页面，包括用户信息、设置、认证等。
- **visualization/**
  - 可视化分析页面，适合做数据分析、图表展示等。

---

> 你可以根据实际业务需求，扩展或调整这些页面结构。
