# 多维数据分析页面说明

本页面聚合了医疗管理系统中多维度的核心数据分析图表，帮助用户从不同角度洞察患者特征、诊疗行为和用药模式。

## 1. 年龄维度分析（AgeDimensionChart）

- **作用**：展示不同年龄段患者的分布情况。
- **可洞察**：可识别高发年龄段，为疾病预防、健康宣教和资源配置提供依据。

## 2. 地域维度分析（RegionDimensionChart）

- **作用**：展示患者在不同地区（如省、市、区）的分布。
- **可洞察**：发现疾病的地理高发区，辅助流行病学分析和区域医疗资源调配。

## 3. 职业维度分析（JobDimensionChart）

- **作用**：统计不同职业人群的疾病分布及类型关系。
- **可洞察**：分析职业暴露与疾病的相关性，为职业健康管理和政策制定提供数据支持。

## 4. 时间维度分析（TimeDimensionChart）

- **作用**：展示患者数量随时间的变化趋势。
- **可洞察**：识别疾病发病高峰期、季节性波动，为医疗资源调度和预警提供参考。

## 5. 分期-药品用药流向分析（TreatmentSankey）

- **作用**：通过桑基图展示不同疾病分期患者的主要用药流向。
- **可洞察**：揭示各分期常用药品组合，辅助临床路径优化、用药规范化和药品采购决策。

---

> 通过上述多维度分析，用户可以全方位了解患者群体特征、诊疗行为模式及用药趋势，为临床决策、管理优化和科研分析提供有力的数据支撑。
